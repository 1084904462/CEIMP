欢迎使用综合测评管理系统

综测信息填写

2017/11/11，综测信息填写开始。填写前请先阅读 《综合测评审核与填写规则》。

综测审核流程

一审

2017/11/15，综测信息填写截止，进入一审阶段，请各位班长作好各班的综测审核工作。如有发现填写错误，请通知同学进行修改，或由班长代替修改。

二审

2017/11/18，一审截止，由辅导员进行二审。

二审错误修改

2017/11/22，二审结束，辅导员将二审中出现的错误反馈给各班班长，通知同学进行修改，或由班长代替修改。

审核结束后续

查看公示结果

2017/11/28,审核结束公示综测结果。请各位同学及时查看成绩，如有疑问，请联系各班班长。

奖学金申报

2017/11/30，开始各项奖学金申报，请参照奖学金填写模板进行奖学金表格填写。
